# Tower Site Analysis Tool

This tool generates **site inspection dashboards and maps** for cellular tower planning using data from an Excel spreadsheet.

It automatically gathers:

- Google Street View imagery
- 360° inspection views
- Satellite maps
- Road maps
- Coverage radius overlays
- Area classification (Urban / Suburban / Rural)
- Automated tower detection

The results are exported as **HTML dashboards** that can be zipped and shared with clients.
